# 💕 Do You Still Love Me? — Standalone Project

A cute interactive love question page with a growing YES button. When she says YES → she sees **Happy 43rd Monthsary, My Love! I love you so much 💕**

## 🌷 How It Works

- **NO button** → shrinks & fades every click, while the **YES** button grows bigger and bigger, taking over more of the screen.
- After **6 NO clicks**, YES has grown to fill the screen and it jumps straight to the celebration page — same as clicking YES directly.
- **YES button** (at any point) → leads to the **Happy 43rd Monthsary** page with confetti, a giant "YES!" badge, and "I knew it 🌸" 🎉

## 🚀 Publish to GitHub Pages

1. Create a new repo (e.g. `love-question`)
2. Upload `index.html` and `style.css`
3. **Settings → Pages → Deploy from branch → main → / (root)**
4. Live at: `https://yourusername.github.io/love-question`
