# 💕 Do You Still Love Me?

A cute, self-contained "Do you still love me?" page. Say **YES** and you're taken straight to a **Happy 43rd Monthsary** celebration page with confetti.

## 🌷 How It Works

- **NO button** → shrinks and fades a little more with every click, while the **YES** button grows bigger and bigger.
- Keep clicking NO and YES eventually grows to **take over the entire screen** — there's no automatic redirect, you still have to actually tap YES.
- **YES button** (at any point) → the **Happy 43rd Monthsary** celebration page, with falling confetti.

## 🚀 Publish to GitHub Pages

1. Create a new repo (e.g. `love-question`)
2. Upload `index.html`, `style.css`, and the `assets/` folder
3. **Settings → Pages → Deploy from branch → main → / (root)**
4. Live at: `https://yourusername.github.io/love-question`
